# 📧 Email Template Gallery

A collection of responsive HTML email templates built using **MJML** and **HTML/CSS**, designed to render consistently across major email clients like Gmail, Outlook, and mobile devices.

## 🚀 Live Demo

👉 https://emailtemplategallery.netlify.app/

---

## 📌 About This Project

This project showcases a set of professionally structured email templates commonly used in real-world email marketing, including:

* Welcome Emails
* Promotional Campaign Emails
* Newsletter Layouts
* Transactional Emails (e.g. password reset)

Each template is designed with a focus on:

* ✅ Responsive design
* ✅ Cross-client compatibility
* ✅ Clean and reusable structure
* ✅ Email development best practices

---

## 🛠️ Technologies Used

* MJML (Email Framework)
* HTML5
* CSS3 (Inline styling for email compatibility)
* JavaScript (for gallery functionality)

---

## 💡 Key Features

* 📱 Fully responsive email templates
* 📧 Table-based layouts for reliable rendering
* 🎯 Optimized for Gmail, Outlook, and mobile devices
* 🧩 Reusable email components
* 🌐 Live preview of each email template

## ⭐ Notes

Email development has unique constraints compared to web development. This project highlights techniques such as:

* Using tables instead of divs
* Applying inline CSS
* Testing across multiple email clients
